## English

GridWise is a professional-grade image processing tool designed for creators and social media managers. It combines smart grid splitting, intelligent background removal, and freeform cropping to help you process high-quality image assets in seconds.

### âœ¨ Key Features

- **Smart Grid Splitter**: Supports custom rows and columns (from 1x1 up to 10x10). Perfect for WeChat, Instagram, and Xiaohongshu grid layouts.
- **Intelligent Background Removal**: Built-in color recognition algorithm with adjustable similarity threshold, edge smoothing, and "outer-only" flood-fill logic.
- **Precision Cropping**: High-accuracy cropping tool with real-time preview and Rule of Thirds grid lines.
- **Multi-format Export**: Export as PNG, JPG, or WebP. Download individual tiles or all tiles in a single ZIP file.
- **Bilingual Interface**: Seamlessly switch between Simplified Chinese and English.
- **Superior UX**: Smooth zooming, dragging, and interactive tile selection.

### ğŸ› ï¸ Tech Stack

- **Framework**: React 19 (Hooks)
- **Styling**: Tailwind CSS
- **Image Processing**: HTML5 Canvas API
- **Compression**: JSZip & FileSaver
- **Typography**: Google Fonts (Inter)

### ğŸš€ Quick Start

1. **Upload**: Drag and drop an image or click the upload area.
2. **Edit**:
   - Adjust rows/columns in the **Grid Splitter** tab. Click specific tiles to select them.
   - Pick a background color and adjust threshold in the **Remove Background** tab.
   - Use the **Crop Image** button to refine your composition.
3. **Export**: Select your preferred format (PNG/JPG/WEBP) and click "Download All" or "Download Selected".
